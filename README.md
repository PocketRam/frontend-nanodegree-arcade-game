## Prerequisites

Browser support for JavaScript ES6, HTML5, and CSS3 are required
Bootstrap 4.0

The following browsers support this app:
Google Chrome,
Microsoft Edge,
Mozilla Firefox,
Safari

## Instructions

Move your character with the arrow keys to get across the road. Get to the water to win!
